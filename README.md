# comp3140project
A banking system created for COMP 3140.


How to Run: 
1. To run the application: Go to terminal & enter: g++ programfiles\*.cpp programfiles\ServiceClass\*.cpp programfiles\Repository\*.cpp -Iheaders -std=c++17 -o bank_app.exe

2. Then execute the exe file in the terminal: .\bank_app.exe

The account creation, and a list of account numbers should be visible.
